**Input Visualiser** is a lightweight, real-time keyboard and mouse overlay for Windows that visually displays key presses, mouse clicks, and scroll events on-screen. Built with Python, Tkinter, and Pynput, it provides a clean, transparent, click-through interface designed for streamers, tutorial creators, software demonstrations, presentations, and input monitoring.

The application renders a full keyboard layout with live key highlighting, visual mouse button indicators, scroll activity feedback, input statistics, and global hotkeys for quick control. It supports multiple languages, modifier key tracking, Windows key combinations, always-on-top display mode, and optional administrator relaunch for enhanced input capture.

## Features
   • Real-time keyboard visualization
   
   • Live mouse click and scroll tracking
   
   • Transparent, always-on-top, click-through overlay
   
   • Multi-language support (English, Spanish, French, German, Japanese, Chinese, Arabic)
   
   • Modifier key and shortcut detection
   
   • Input statistics (keys, clicks, scrolls)
   
   • High refresh-rate updates for smooth visual feedback
   
   • Stream-friendly minimal dark theme
   
   • Windows administrator relaunch support
   
   ### Global hotkeys:
   
   • Ctrl + Q --> **Quit**
          
   • Ctrl + D --> **Toggle display**
          
   • Ctrl + Shift + A --> **Relaunch as administrator**
